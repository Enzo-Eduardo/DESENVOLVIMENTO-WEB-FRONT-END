Termos-chave

HTML
HTML significa "HyperText Markup Language" e é a linguagem de marcação usada para estruturar o conteúdo das páginas web.
CSS
CSS significa "Cascading Style Sheets" e é usado para definir o estilo de apresentação de uma página web, como cores, fontes e layout.
JavaScript
JavaScript é uma linguagem de programação usada para tornar as páginas web interativas, permitindo a criação de animações, validação de formulários, entre outros recursos.
